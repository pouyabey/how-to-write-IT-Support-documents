## Device / Asset Details

### 1. Item Name

Write the name of the device or item.

Example:
Laptop, desktop, monitor, printer, scanner, docking station, keyboard, mouse, phone, router, switch.

---

### 2. Asset Tag

Write the company asset tag or inventory number.

Example:
IT-001, LAP-102, MON-204

---

### 3. Serial Number

Write the serial number from the device label or system information.

Example:
CN123456789

---

### 4. Model

Write the device brand and model.

Example:
Dell Latitude 5420, HP EliteDesk 800, Lenovo ThinkPad T14

---

### 5. Assigned User

Write the name of the employee using the device.

Example:
John Smith

---

### 6. Location

Write where the device is physically located.

Example:
Office 204, Front Desk, IT Storage Room, Conference Room

---

### 7. Status

Choose the current status of the item.

Common statuses:

- In Use
- Available
- In Storage
- Under Repair
- Retired
- Lost
- Replaced

---

### 8. Notes

Add any extra information about the item.

Example:
Needs replacement, missing charger, warranty active, assigned temporarily, damaged screen.

## Final Verification

After completing the inventory document, verify that:

- Asset tag is correct
- Serial number is recorded accurately
- Device location is correct
- Assigned user is correct
- Status is updated
- Any missing or damaged items are noted

## Troubleshooting / Common Issues

If there are problems with the inventory record, check the following:

- If the serial number is missing, check the device label or system settings
- If the assigned user is unknown, ask the department supervisor
- If the asset tag is missing, report it to IT or inventory management
- If the device is damaged, update the status and add notes
- If the device cannot be found, mark it as missing and escalate

## Notes

- Update the inventory whenever a device is assigned, moved, repaired, replaced, or retired
- Keep asset records accurate for auditing and tracking
- Escalate missing, damaged, or untagged equipment to the IT supervisor
- Perform regular inventory checks to keep records up to date
